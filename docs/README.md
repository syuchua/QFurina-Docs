# QFurina - QQ机器人项目

欢迎来到QFurina项目！这是一个基于Python的QQ机器人后端服务，提供了一系列自动化和交互功能。

## 主要功能

1. AI聊天与角色扮演
2. 群聊管理
3. 图片获取与发送（支持pixiv）
4. AI绘画
5. AI语音生成
6. 天气查询
7. 联网搜索
8. 定时开关机

## 支持的消息平台

目前主要在Windows端Llonebot上测试，理论上所有支持onebot v11协议的消息平台都可以使用。HTTP对接配置可能需要额外设置。
